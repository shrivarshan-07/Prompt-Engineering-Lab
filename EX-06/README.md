# 🤖 Ex-6 Development of Python Code Compatible with Multiple AI Tools

## 🎯 Aim
To understand how AI-assisted prompting can be used for generating Python code, interacting with APIs, comparing outputs, and producing meaningful insights using multiple AI tools.

---

# 🎯 Objective

Learners will learn how to design effective prompts for AI tools to assist them in coding tasks related to mini and final year projects.

The experiment focuses on:

- 🐍 Writing prompts for Python code generation
- 🔗 API interaction using AI-generated code
- ⚖️ Comparing outputs from multiple AI tools
- 🛠️ Refining prompts for better coding assistance

---

# 🧪 Experiment

Instead of directly writing the code, learners must practice creating prompts that guide AI tools to:

- 🐍 Generate Python code for interacting with multiple APIs.
- 📊 Compare outputs from different APIs and highlight differences.
- 💡 Suggest meaningful insights or next steps based on the generated results.

---

# 📝 Algorithm

## Step 1: Define the Problem Statement

- 🎯 Choose a simple project idea or coding task.
- 🔗 Identify the APIs or datasets to be used.

---

## Step 2: Design Effective Prompts

- ✍️ Write prompts for generating Python code.
- 📌 Frame prompts clearly with expected outputs.

---

## Step 3: Generate AI Responses

Use AI tools such as:

- 🤖 ChatGPT
- 💎 Gemini
- 🧠 Claude

Generate Python programs and explanations.

---

## Step 4: Compare Outputs

- ⚖️ Compare responses generated by multiple AI tools.
- 🔍 Analyze differences in code quality and explanations.

---

## Step 5: Refine Prompts

- 🛠️ Modify prompts for better results.
- ✨ Improve clarity and specificity.

---

## Step 6: Document the Results

- 📄 Add generated prompts
- 🐍 Include AI-generated code
- 📸 Add screenshots and outputs
- 📝 Write observations and insights

---

# 🎯 Output

Learners should generate:

- 🐍 Python code outputs
- 🔗 API interaction examples
- 📊 Comparative analysis of AI-generated responses
- ✨ Refined prompts and improved outputs
