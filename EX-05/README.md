# 🤖 Ex-5 Comparative Analysis of Different Types of Prompting Patterns and Various Test Scenarios

## 🎯 Aim
To understand and compare different prompting techniques such as Basic Prompting, Zero-shot Prompting, Few-shot Prompting, and Chain-of-Thought Prompting by applying them to various real-world scenarios using AI tools.

---

# 🧪 Experiment

Test and compare how different prompting models respond to various prompts (broad or unstructured) versus refined prompts across multiple scenarios using:

- 🟢 Zero-shot Prompting
- 🧩 Few-shot Prompting
- 🧠 Chain-of-Thought Prompting

---

# 📌 Activity

- ✍️ Start with a basic prompt.
- 🔄 Convert it into a zero-shot prompt.
- 📚 Extend it into a few-shot prompt with at least two examples.
- 🧠 Rewrite it into a chain-of-thought prompt where reasoning is shown step by step.
- ⚖️ Compare the outputs generated by different prompting methods.

---

# 🎯 Objective

Learn how the same task can be refined into multiple prompting techniques and analyze the quality, accuracy, and depth of generated responses.

---

# 📝 Algorithm

## Step 1: Select a Real-World Scenario

Choose any one scenario such as:

- 🛍️ Product recommendation
- 📧 Email generation
- 📄 Report writing
- ✈️ Travel planning
- 💬 Customer support chatbot
- 🎓 Educational assistant

---

## Step 2: Create a Basic Prompt

Write a simple prompt without detailed instructions.

### 📌 Example

```text
Write an email to a customer about delayed delivery.
```

---

## Step 3: Convert into Zero-shot Prompt

Rewrite the prompt clearly without giving examples.

### 📌 Example

```text
Generate a professional apology email to a customer regarding delayed product delivery.
```

---

## Step 4: Create Few-shot Prompt

Provide at least two example inputs and outputs before asking the final question.

---

## Step 5: Create Chain-of-Thought Prompt

Rewrite the same prompt with step-by-step reasoning instructions.

### 📌 Example

```text
Think step by step and generate a professional response explaining the reason for delivery delay and compensation details.
```

---

## Step 6: Generate Outputs Using AI Tools

Use minimum two AI tools:

- 🤖 ChatGPT
- 💎 Gemini
- 🧠 Claude
- 🪟 Copilot

Capture outputs and screenshots.

---

## Step 7: Compare Responses

Analyze:

- ✅ Clarity
- 🎯 Accuracy
- 📚 Completeness
- 🎨 Creativity
- 🌟 Response quality

Prepare a comparison table.

---

## Step 8: Prepare Final Report

Include:

- 🎯 Aim
- 📌 Objective
- 🧠 Prompt types
- 💬 Example prompts
- 📥 Outputs
- 📊 Comparison table
- ✅ Result

Export report as PDF.

---

# 🎯 Output

The AI tools generate different responses based on the prompting techniques used. Structured prompts improve clarity, reasoning, and response quality.

---

# ✅ Result

Thus, different prompting techniques such as Zero-shot, Few-shot, and Chain-of-Thought prompting were implemented and analyzed successfully using multiple AI tools.
