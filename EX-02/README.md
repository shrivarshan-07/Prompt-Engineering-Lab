# 🌐 Ex-2: Cross Platform Prompting – Evaluating Diverse Techniques in AI Powered Text Summarization

## 🎯 Aim
To understand and evaluate different prompting techniques across multiple AI platforms by performing text summarization and comparing the outputs.

---

# 🧪 Experiment

Perform text summarization using different prompting techniques on multiple AI platforms such as:

- 🤖 ChatGPT
- 💎 Gemini
- 🪟 Copilot

Use the same input text and evaluate the outputs based on different criteria.

---

# 📝 Algorithm

1. 📄 Select a common paragraph for summarization.
2. 🌐 Open AI platforms such as ChatGPT, Gemini, and Copilot.
3. 📥 Provide the same input text to all platforms.
4. 🧠 Apply different prompting techniques:
   - Zero-shot prompting
   - Few-shot prompting
   - Chain-of-thought prompting
   - Role-based prompting
5. 🖥️ Record the outputs generated by each platform.
6. 📊 Compare the outputs based on:
   - Accuracy
   - Coherence
   - Simplicity
   - Speed
   - User Experience
7. 🔍 Analyze the differences between outputs.
8. 📋 Prepare a comparison table.
9. 📝 Document observations and conclusion.
10. 📄 Convert the report into PDF format and submit.

# 💬 Prompting Techniques

## 1️⃣ Zero-Shot Prompting
📌 *Summarize the given text without providing any examples.*

### Example Prompt
```text
Summarize the following paragraph in simple words.
```

---

## 2️⃣ Few-Shot Prompting
📌 *Summarize the text by including 2–3 examples.*

### Example Prompt
```text
Example 1:
Input: Artificial Intelligence improves automation.
Summary: AI helps automate tasks.

Example 2:
Input: Cloud computing provides online storage.
Summary: Cloud computing stores data online.

Now summarize the following paragraph.
```

---

## 3️⃣ Chain-of-Thought Prompting
📌 *Generate a detailed explanation based on previous summaries.*

### Example Prompt
```text
Read the paragraph carefully.
Identify the main idea.
Break the paragraph into key points.
Then generate a detailed summary.
```

---

## 4️⃣ Role-Based Prompting
📌 *Summarize the text by assuming a specific role.*

### Example Prompt
```text
Act as a researcher and summarize the following paragraph professionally.
```

---

# 📊 Evaluation Criteria

Evaluate outputs based on:

| 📌 Criteria | 📝 Description |
|---|---|
| ✅ Accuracy | Correctness of content |
| 🔗 Coherence | Logical flow of ideas |
| 😊 Simplicity | Ease of understanding |
| ⚡ Speed | Response time |
| 🌟 User Experience | Ease of use and interaction |
