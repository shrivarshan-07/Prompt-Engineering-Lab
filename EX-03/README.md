## 🎯 Aim
To design and implement prompt engineering techniques for building an AI-powered chatbot that assists users in product troubleshooting, order tracking, and general inquiries.

---

# 🧪 Experiment

Explain and demonstrate the following Prompt Engineering techniques with examples:

- 🟢 Straightforward Prompts
- 📊 Tabular Format Prompting
- 🧩 Missing Word Prompting
- ❓ Preceding Question Prompting

---

# 📝 Algorithm

1. 🎯 Identify the chatbot use case (customer support).
2. 🧠 Select different prompt engineering techniques.
3. ✍️ Design prompts for each technique.
4. 🤖 Execute prompts using AI tools (ChatGPT / Gemini).
5. 📥 Analyze the outputs generated.
6. ⚖️ Compare responses for effectiveness.
7. 📄 Document examples and outputs.
8. 🧾 Prepare report and convert to PDF.
9. ☁️ Submit through GitHub.

---

# 💡 Prompting Techniques with Examples

## 1️⃣ Straightforward Prompt

### 📌 Prompt

```text
Explain how to track an online order.
```

### 🤖 Output

The AI provides a direct and clear answer explaining steps to track an order.

---

## 2️⃣ Tabular Format Prompting

### 📌 Prompt

```text
Provide customer issues and solutions in a table format.
```

### 🤖 Output

| Issue | Solution |
|---|---|
| Order delay | Check tracking ID |
| Product defect | Request replacement |

---

## 3️⃣ Missing Word Prompting

### 📌 Prompt

```text
Fill in the blank: The order status can be checked using ______.
```

### 🤖 Output

```text
tracking ID / order ID
```

---

## 4️⃣ Preceding Question Prompting

### 📌 Prompt

```text
Before answering, ask what the issue is. Then provide a solution.
```

### 🤖 Output

AI first asks a question, then responds accordingly.
